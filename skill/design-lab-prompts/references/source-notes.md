## Coverage

The current source contains 77 styles, 174 Chinese prompt entries, and 144 English prompt entries. The original 67 styles have bilingual metadata. Ten newer styles—`cream`, `grain`, `editorialsaas`, `ambient`, `spec`, `ascii`, `victorian`, `atomic`, `zine`, and `halftone`—are Chinese-only in the captured source.

Search-engine cache text still advertised 67 styles and 130+ prompt templates during capture. The local catalog uses the current rendered source, records the complete 77-style inventory, and exposes the cache drift rather than silently matching the stale counter.

The source recommender is also preserved: 12 project categories × 4 creativity levels × 3 recommendations, or 144 recommendation records.

## Text fidelity

Prompt text, labels, CSS snippets, descriptions, characteristics, palettes, and do/don't rules are stored as exposed by the source. Normalization changes object structure and escaping only. No missing English text is machine-translated.
